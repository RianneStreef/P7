# Connect - E
Connect-E is a project for the Web Developer course from OpenClassrooms. The goal is the create a MVP for a internal social network to share articles. 

## Tech/framework used
React - Node.js

## Installation
Front end - 
```
npm init
npm run start
```

Back end - 
```
npm init 
npm run dev
```
